## 🚀 如何打开报告

### 方法1: 使用浏览器直接打开
```bash
# 在项目根目录执行
open visual-test-results/final-visual-report.html
open comprehensive-test-results/comprehensive-report.html

# 或者双击HTML文件用浏览器打开
```

### 方法2: 使用VS Code Live Server (推荐)
1. 在VS Code中安装Live Server扩展
2. 右键点击HTML文件 → "Open with Live Server"
3. 自动在浏览器中打开

### 方法3: 使用Python简单服务器
```bash
# 在项目根目录执行
python3 -m http.server 8000
# 然后在浏览器访问:
# http://localhost:8000/visual-test-results/final-visual-report.html
# http://localhost:8000/comprehensive-test-results/comprehensive-report.html
```

## 🔧 如果报告打不开

### 常见问题解决
1. **浏览器安全限制**: 某些浏览器可能阻止本地HTML文件，请使用Live Server
2. **文件路径问题**: 确保在正确的项目根目录
3. **图片显示问题**: 部分图片可能需要相对路径，这是正常的演示状态

### 重新生成报告
```bash
# 如果报告有问题，可以重新生成
node generate-demo-reports.js
```
